## 🎬 The 60-Second Tour

### What This App Does
A fun, animated racing game that randomly picks someone to buy lunch!

### How It Works
1. Add names and emojis
2. Select 2+ people
3. Click "Start Race"
4. Watch cute animation
5. Random winner = buys lunch! 😄

### Key Features
- 🏃 Cute running animations
- 🎉 Celebration effects
- 💾 Data saves automatically
- 📱 Works on phones
- 🎨 Beautiful design
- 🔗 Share with URL parameters
